# ד״ר יען יעקב סופר — Cyber & Digital Intelligence

Professional one-page site for **Dr. Yaen Yaacov Sofer** — cyber researcher and digital
intelligence expert (OSINT · SOCMINT · Dark Web · Cyber Threat Intelligence), lecturer at
Bar-Ilan University.

Static site, Hebrew (RTL). No build step.

## Files
- `index.html` — markup
- `styles.css` — design system (dark, intelligence-grade theme)
- `script.js` — nav, scroll reveal, contact form

## Run locally
Any static server, e.g.:

```bash
python -m http.server 4787
# then open http://localhost:4787
```

## Deploy
Served via GitHub Pages from the `main` branch root. Any static host (Netlify, Vercel,
Cloudflare Pages) works by dropping these files in.
